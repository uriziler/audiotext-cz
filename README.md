# AudioText CZ

AudioText CZ je připravovaná Android aplikace pro lokální přepis audio nahrávek do textu. MVP cílí na Samsung S25 Ultra, češtinu jako výchozí jazyk a offline přepis pomocí lokálních Whisper modelů.

## MVP

- Import existující audio nahrávky.
- Lokální přepis bez cloudu.
- Čeština jako defaultní jazyk.
- Modely Base, Small, Medium a Large.
- Uložení přepisu po časovaných segmentech.
- Editace přepsaného textu.
- Export do TXT.
